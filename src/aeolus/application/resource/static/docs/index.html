Aeolus API Dokumentation

Willkommen zur API-Dokumentation von Aeolus. Unsere API ermöglicht es Ihnen, Temperaturdaten einfach und flexibel zu erfassen, zu analysieren und in Ihre eigenen Anwendungen zu integrieren. Diese Dokumentation bietet Ihnen alle notwendigen Informationen, um die API optimal zu nutzen.

Authentifizierung

Um die API nutzen zu können, müssen Sie sich authentifizieren. Verwenden Sie hierzu einen API-Schlüssel, den Sie in Ihrem Aeolus-Konto generieren können.
Beispiel für die Authentifizierung: 
GET /rest/readings/2024
Hades-Login-Token: YOUR_API_KEY
